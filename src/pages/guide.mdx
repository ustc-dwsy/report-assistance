# 🔬 物理实验数据分析辅助工具

本工具专为**大学物理实验**的误差分析环节设计，旨在通过**对数微分法**快速、准确地计算间接测量量的不确定度。


**核心方法**：对数微分法——对函数先取自然对数，再求全微分，从而高效推导相对不确定度公式。


## 📖 使用指南

### 第一步：准备计算公式
确定你的间接测量量与各直接测量量之间的函数关系。
*   例如测量圆柱体密度：`ρ = 4M / (π·D²·H)`

### 第二步：按语法输入表达式
在输入框中键入公式。请严格遵守以下语法规则：

| 类别 | 支持项 | 输入示例 |
| :--- | :--- | :--- |
| **基本运算** | 加、减、乘、除、幂 | `a + b`, `x / y`, `r^2` |
| **单目函数** | 平方根、三角函数等 | `sqrt(x)`, `sin(θ)`, `ln(y)` |
| **内置常量** | 圆周率、自然常数 | `pi`, `e` |


**注意**：乘号必须显式写出（`*`），例如 `2 * x` 而非 `2x`。幂运算请使用 `^`。


### 第三步：获取结果
工具会自动应用对数微分法，推导并展示该公式对应的**不确定度传递公式**。

## ℹ️ 关于工具

### 实现原理
本工具的核心是一个基于符号计算的**表达式解析引擎**。当用户输入公式后，引擎会执行以下步骤：
1.  **解析**：将字符串公式转换为抽象语法树（AST）。
2.  **微分**：遍历语法树，根据对数微分法则对目标变量求偏导。
3.  **合成**：将各项偏导平方求和，构建出完整的相对不确定度公式。

### 适用范围与局限性
*   **适用**：适用于大学物理实验中，各直接测量量**相互独立**且误差服从正态分布的间接测量。
*   **局限**：目前版本暂不支持分段函数、统计回归分析以及非独立变量的协方差项计算。
